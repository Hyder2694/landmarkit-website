# LandmarkIT Website

This package is ready for GitHub Pages.

## Files to upload
Upload everything inside this folder to your GitHub repository:
- index.html
- about.html
- services.html
- staffing.html
- software-development.html
- careers.html
- contact.html
- assets folder

## GitHub Pages setup
1. Create a new GitHub repository, for example: landmarkit-website
2. Upload all files from this package
3. Go to Settings > Pages
4. Source: Deploy from a branch
5. Branch: main
6. Folder: /root
7. Save

## Custom domain
After the site works on GitHub Pages, connect:
landmarkitstaffing.com

The contact form is intentionally skipped. Contact uses direct email and phone links.
